## рҹ“Ң How to Use

1. Select the appropriate **branch (syllabus year)**  
2. Open your respective **semester folder**  
3. Choose either:
   - `Question_Papers`
   - `Study_Materials`
4. Select the required academic year folder  
5. Download the material
